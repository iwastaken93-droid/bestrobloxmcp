# BestRobloxMCP VSCode Extension

> Companion extension for BestRobloxMCP — live Studio tree, one-click sync, inline script editing.

## Features (Planned)

- **Live Studio Tree** — View connected Studio instances in the VSCode sidebar
- **One-Click Sync** — Push/pull files between Studio and local workspace
- **Inline Script Editing** — Edit Luau scripts with auto-save
- **Screenshot Preview** — View captured screenshots in a panel

## Setup

```bash
cd vscode-extension
npm install
vsce package
```

## Install

Install the `.vsix` file via VSCode:
```
Cmd+Shift+P → Extensions: Install from VSIX
```

## Development

Open this folder in VSCode and press F5 to launch the Extension Development Host.
